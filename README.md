# IoT Monitoring System

A simple IoT monitoring system that simulates sensor data, stores it in a SQLite database,
and visualizes it using a web-based dashboard.

## Features
- Simulated temperature and humidity sensor
- SQLite database storage
- Streamlit web dashboard
- Data visualization over time

## Technologies
- Python
- SQLite
- Streamlit
- Google Colab

## Files
- `app.py` – Streamlit dashboard
- `iot_data.db` – SQLite database
- `.ipynb` – Google Colab notebook

## Note
Sensor data is simulated due to platform limitations.
